Калькулятор для выполнения базовых вычислений (сложение, вычитание, умножение, деление, возведение в степень, извлечение корня). Также добавлены кнопки для удобства, чтобы калькулятор не был зависимым от клавиатуры.

Технологии: .NET 8.0 Avalonia UI - кросс-платформенный UI-фреймворк ReactiveUI - фреймворк для реактивного программирования MVVM Pattern - Model-View-ViewModel Autofac - DI-контейнер (в основном проекте)
